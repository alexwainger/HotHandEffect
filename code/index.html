Views vs. Conversion Rate Scatter Plot

Author: Alex Wainger, Login: awainger

This visualization aims to illustrate the conversion rate and number of views, grouped by different pieces of information. For example, by selecting the "city" option from the drop down menu, you can see each city in the data set plotted with view counts on the X axis and conversion rate (fund events / view events) on the Y axis. The X-axis uses a power scale of .1 so that the data at the smaller end of the spectrum is more spaced out.

There is also an "average conversion rate" line drawn, which shows the average conversion rate for the entire dataset (around 70%) and serves as a reference to compare the data points with. A tooltip will also appear if you hover over a data point, which gives you the name of the group, the view count, and the conversion rate, rounded to three decimal points.

Bootloader should be able to use this visualization to identify which demographics view the most/fewest projects, and which demographics turn a view into a fund most frequently. This should help them focus their marketing strategies on taking advantage of their most productive demographics, or trying to reach their underserved ones.
